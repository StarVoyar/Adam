# Adam — A Minimal LLM and AI From Scratch

Adam is a fully custom, from‑scratch implementation of a small language model pipeline.  

It includes:
- A BPE tokenizer trained on multiple datasets  
- A dataset builder with section‑aware splitting  
- A GPT‑style Transformer decoder  
- A full test suite  
- A checkpointing system  
- A clean project structure designed for experimentation

This project is built and tested on Arch Linux with Python 3.11+.
